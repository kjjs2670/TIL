# 1/11 파이썬 챗봇 만들기(파이썬 기초)

---



### 컴파일 언어와 인터프리터 언어

---

- C, C++, Java : 컴파일 언어

- Python, JavaScript, Ruby : 인터프리터 언어

- 소스파일을 컴파일러로 번역 시 .obj 파일이 생성됨.

      여러개의 obj 파일을 하나의 실행파일로 합쳐주는 링커가 있음.

      한꺼번에 번역된 것이 한번에 실행되므로 실행속도가 빠름.

- 인터프리터는 실행 시 소스파일이 인터프리터로 넘어가 번역, 실행을 같이 함.

  수정이 간단함. 유지보수가 편함. 실행속도가 느림.



### 파이썬의 문법

---

1. 저장 (변수, 리스트, 딕셔너리)

2. 조건
   
   * if : 조건이 참이면 실행
   
   * else : else문 바로 위에 조건이 거짓이면 실행
   
   * elif : 바로 위에 조건이 거짓이고 내 조건(elif)이 참이면 실행

3. 반복
   
   - 가독성 : for
   
   - 무한루프 : while

4. 함수
   
   - 함수 정의(def 함수이름 (): ) & 함수 호출
   
   - 내장함수, 외장함수(import), 사용자함수
